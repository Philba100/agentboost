**What it does:** Scores leads using BANT/MEDDIC frameworks (0-100 qualification score)

## 💡 Pro Tips for Best Results

1. **Be specific** - More details = better results
2. **Provide numbers** - "Budget: $500k" beats "Large budget"
3. **Set timeline** - "Q2 deadline" matters for scheduling
4. **Request format** - "Give me CSV format" or "Make it a list"
5. **Ask for alternatives** - "Show me 3 options"

## 🔗 Get Started

1. Copy your skill's share link from the dashboard
2. Open Edge Gallery → Tap Skills icon
3. Tap (+) → "Load skill from URL"
4. Paste share link
5. Start chatting!
